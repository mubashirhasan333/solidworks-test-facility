# SolidWorks Experimental Facility Design

## Overview

This project highlights CAD and mechanical design work for experimental facility components, test-section layouts, and supporting mechanical systems.

## Engineering Problem

Experimental research facilities require practical mechanical designs that support repeatable testing, safe operation, manufacturability, assembly, instrumentation access, and clear communication between researchers, fabricators, and technicians.

## Methods

- Developed 3D CAD models for experimental facility components
- Prepared assemblies and layout views for communication
- Considered fabrication, accessibility, support structures, and instrumentation constraints
- Created engineering visuals and documentation for review

## Tools

- SolidWorks
- ANSYS SpaceClaim
- Siemens NX training
- Engineering drawings and layout interpretation

## Skills Demonstrated

- Mechanical design
- CAD modeling
- Assembly planning
- Design-for-fabrication awareness
- Engineering communication
